# Myeloid Cell Annotation Guide

## Overview

Myeloid cells are a diverse group of immune cells derived from myeloid progenitors in the bone marrow. They include monocytes, macrophages, dendritic cells, and granulocytes.

## Key Canonical Markers

### Pan-Myeloid Markers
- **CD33**: Myeloid lineage marker
- **ITGAM (CD11b)**: Integrin alpha M
- **CSF1R (CD115)**: Colony stimulating factor 1 receptor

### Monocyte/Macrophage Markers
- **CD14**: Classical monocyte marker
- **CD68**: Pan-macrophage marker
- **FCGR3A (CD16)**: Non-classical monocyte marker

### Dendritic Cell Markers
- **HLA-DRA/HLA-DRB1**: MHC class II
- **ITGAX (CD11c)**: Dendritic cell marker
- **CLEC9A**: cDC1 marker
- **CD1C**: cDC2 marker

## Subtype Differentiation

### Classical Monocytes
**Key markers**: CD14++, CD16-, S100A8/A9+, VCAN+
**Function**: Phagocytosis, inflammation

### Non-classical Monocytes
**Key markers**: CD14+, CD16++, FCGR3A+
**Function**: Patrolling, tissue repair

### Intermediate Monocytes
**Key markers**: CD14++, CD16+
**Characteristics**: Transitional phenotype

### Macrophages
**Key markers**: CD68+, CD163+, MARCO+, MRC1 (CD206)+
**Note**: Highly tissue-dependent marker profiles
**M1 (inflammatory)**: IL1B+, TNF+, NOS2+
**M2 (anti-inflammatory)**: CD163+, MRC1+, ARG1+

### Dendritic Cells
**cDC1**: CLEC9A+, XCR1+, BATF3+ (cross-presentation)
**cDC2**: CD1C+, FCER1A+ (T cell activation)
**pDC**: LILRA4+, IRF7+, CLEC4C+ (type I interferon)

## Common Pitfalls

1. **CD14 variability**: CD14 expression varies with activation state
2. **Macrophage plasticity**: M1/M2 is a spectrum, not discrete states
3. **Tissue macrophage diversity**: Each tissue has distinct macrophage populations
4. **DC contamination**: DCs can be confused with activated monocytes

## Technical Notes

- Monocyte/macrophage distinction can be challenging in tissues
- Use tissue context to interpret macrophage phenotypes
- DCs are relatively rare and may cluster with other myeloid cells
