# CASSIA 0.1.0

Initial CRAN release.

## Features

* Core annotation functions: `runCASSIA()`, `runCASSIA_batch()`, `runCASSIA_pipeline()`
* Advanced analysis: `runCASSIA_annotationboost()`, `runCASSIA_subclusters()`
* Multiple LLM providers: Anthropic, OpenAI, OpenRouter
* Uncertainty quantification with `runCASSIA_n_times()`
* Annotation merging with `runCASSIA_merge_annotations()`
* Score reports with `runCASSIA_generate_score_report()`
* Model configuration with `list_models()`, `get_recommended_model()`
* Built-in marker datasets with `loadBuiltinMarkers()`, `listAvailableMarkers()`
* Seurat integration with `add_cassia_to_seurat()`
