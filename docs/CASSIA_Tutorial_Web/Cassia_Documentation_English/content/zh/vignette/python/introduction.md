欢迎来到 CASSIA Python 教程部分。在这里，我们提供使用真实数据的分步教程，帮助您了解 CASSIA 的完整工作流程和功能。

## 可用教程

1. **[基本注释](basic-annotation.md)** - 了解如何使用标记文件通过 CASSIA 注释细胞聚类。这涵盖了安装、设置、快速模式、批量分析和质量评分。

2. **[聚类和注释](clustering-and-annotation.md)** - 从原始数据开始的完整工作流程，使用 Scanpy。了解如何对单细胞数据进行聚类并在无缝管道中使用 CASSIA 对其进行注释。

3. **[扩展分析](extended-analysis.md)** - 了解如何使用 CASSIA 的可选智能体。这涵盖了不确定性量化、注释增强、RAG、细胞类型比较和亚群聚类分析。

## 入门

为了充分利用这些教程，请确保您具备：

1. 正确安装 CASSIA（请参阅 [安装 CASSIA](/docs/python/setting-up-cassia)）
2. 对 Python 编程有基本的了解
3. 熟悉单细胞 RNA 测序概念
