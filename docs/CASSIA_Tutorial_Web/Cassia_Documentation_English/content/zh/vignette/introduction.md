欢迎来到CASSIA教程部分。在这里，我们提供基于真实数据的分步教程，帮助您全面了解CASSIA的完整工作流程和功能。

## 可用教程

我们的教程按复杂程度递增排列：

1. **使用标记文件进行基础注释** - 学习如何在已有标记基因的情况下使用CASSIA对细胞簇进行注释。

2. **使用可选代理的扩展分析** - 学习如何使用CASSIA的可选代理功能。

3. **从原始Seurat对象开始的聚类和注释** - 完整演示scRNA-seq数据的聚类和使用CASSIA进行细胞类型注释的流程。

4. **包含详细质量控制的完整工作流最佳实践** - 高级分析流程，包含全面的质量控制步骤以确保结果的可靠性。*注意：此教程仍在开发中，将在未来更新中扩展。*

## 开始之前

为了充分利用这些教程，请确保您已完成以下准备：

1. 正确安装CASSIA（参见[安装配置](/zh/docs/setting-up-cassia)）
2. 具备R编程基础
3. 熟悉单细胞RNA测序概念和Seurat

如果您对新增教程或改进现有教程有任何建议，欢迎通过我们的[GitHub仓库](https://github.com/ElliotXie/CASSIA)贡献或联系开发团队。
