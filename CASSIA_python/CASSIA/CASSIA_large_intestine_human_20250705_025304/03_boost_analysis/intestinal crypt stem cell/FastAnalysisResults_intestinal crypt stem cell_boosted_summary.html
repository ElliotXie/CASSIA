CASSIA Cell Type Annotation Summary (Breadth-First Analysis)

Single-cell RNA-seq Analysis Report

Overview

This report summarizes the iterative gene expression analysis performed by CASSIA to annotate a specific cell population. The analysis involved one main iteration of hypothesis testing, building upon an initial assessment. A total of 17 genes were examined across all evaluations.

Initial Assessment

The initial cell type hypothesis was LGR5+ ASCL2+ Crypt Base Columnar (CBC) Stem Cells, supported by the expression of canonical markers LGR5, ASCL2, and TDGF1. However, ambiguity arose due to high pseudogene expression and the absence of explicit proliferation markers like MKI67 and PCNA. The lack of TA progenitor markers (e.g., CDK1) and +4 reserve stem cell markers (e.g., BMI1) strengthened the CBC hypothesis, but a mixed or transitional state could not be ruled out without further validation.

Final Annotation

**Final Cell Type**: Crypt Base Columnar (CBC) Stem Cells (LGR5+ ASCL2+). **Confidence**: High. **Key Markers**: LGR5, ASCL2, TDGF1, OLFM4, SMOC2, AXIN2. **Uncertainties Resolved**: Proliferation confirmed (MKI67+), no TA/+4 contamination, no differentiated lineage markers detected.

Key Marker Genes

**Canonical CBC Markers**: LGR5, ASCL2, TDGF1 **Additional CBC Markers**: OLFM4, SMOC2 **Proliferation Markers**: MKI67, PCNA **Wnt Activity Marker**: AXIN2

Recommendations

Spatial validation using techniques like LGR5 RNAscope could further solidify the confidence in the annotation.
